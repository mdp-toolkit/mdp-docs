mdp.configuration._version_too_old(version, known_good)[source]¶

Return True iff a version is smaller than a tuple of integers.

This method will return True only if the version string can confidently be said to be smaller than known_good. If the string cannot be parsed as dot-separated-integers, None (which is false) will be returned.

The comparison is performed part by part, the first non-equal one wins.

>>> _version_too_old('0.4.3', (0,4,3))
False
>>> _version_too_old('0.4.2', (0,4,3))
True
>>> _version_too_old('0.5.devel', (0,4,3))
False
>>> _version_too_old('0.4.devel', (0,4,3))
